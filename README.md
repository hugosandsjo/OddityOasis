# Oddity Oasis

A website for the bored. Get todays fun fact or a random fact. After a few buttonclicks the site will suggest an activity

# Installation

1. Clone repository locally


   ```bash
   cd OddityOasis
   npm install
   npm run dev
   ```

# Deploy
   https://oddity-oasis.vercel.app/
